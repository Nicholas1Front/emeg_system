Ajuda com o controle de clientes

Bem-vindo à página de ajuda!

Aqui você encontrará um guia completo para utilizar o sistema de controle de clientes de forma correta e eficiente.

O que é ? Como funciona ?

O sistema de controle de clientes EMEG auxilia na organização e gerenciamento de dados dos clientes e seus equipamentos. Ele funciona de forma local em seu computador, mas requer conexão com a internet para acessar e sincronizar as informações com a nuvem. Vale lembrar que esta plataforma trabalha em conjunto com o sistema de orçamentos, então o cuidado no manuseio dos dados é essencial.

Preenchimento correto

Para manter a padronização e garantir a integridade dos dados, é importante que o usuário siga as instruções para o preenchimento correto dos campos. Preenchimentos incorretos podem gerar erros e até comprometer o envio de informações ao servidor.

Utilize o campo de comentários para identificar possíveis inconsistências nos dados e, se necessário, corrija antes de enviar. Isso ajuda a manter o sistema atualizado e confiável.

Estrutura p/ Cliente

NOME DO CLIENTE + (CONTATO DO CLIENTE / NOME DO CONTATO)

Estrutura p/ Equipamento

TIPO + MARCA + MODELO + PLACA DE IDENTIFICAÇÃO + OUTRA IDENTIFICAÇÃO (FROTA, PLAQUETA, Nº IDENTIF...)

O sistema permite quatro principais ações para gerenciar os dados:

Adicionar

Com os botões "Adicionar cliente" e "Adicionar equipamento", você pode incluir novos dados no sistema com facilidade.

Excluir

Os botões "Excluir cliente" e "Excluir equipamento" permitem remover dados indesejados do sistema de forma prática.

Editar

Use os botões "Editar cliente" e "Editar equipamento" para atualizar informações já cadastradas no sistema.

Enviar

Após realizar as alterações necessárias, clique em "Enviar ao servidor" para atualizar os dados na nuvem.

Passo a passo básico

Para garantir o funcionamento correto do sistema, é essencial utilizá-lo conforme planejado. A seguir, apresentamos um passo a passo simples para orientar o uso adequado da plataforma.

Abra o programa

Antes de iniciar, certifique-se do que precisa ser feito. Por exemplo, se for adicionar um equipamento, verifique as informações necessárias para evitar erros no cadastro. O mesmo cuidado vale para as outras ações, como editar ou excluir.

Realize as alterações necessárias

Com as informações organizadas, faça as alterações no sistema, sempre prestando atenção aos detalhes para evitar a inserção de dados incorretos.

Envie os dados ao servidor

Após finalizar as alterações, clique em "Enviar ao servidor" para sincronizar as informações com a nuvem.

Aguarde o processamento

Depois de confirmar a senha, o sistema precisará de um tempo para processar as informações. Não feche a página até que o processo seja concluído. Uma vez finalizado, você poderá sair com segurança.

Confirmação por senha

Ao realizar ações como "Adicionar", "Editar", "Excluir" ou "Enviar ao servidor", o sistema solicitará uma confirmação de senha para garantir a segurança e evitar alterações acidentais.

Lembre-se: este sistema serve para gerenciar dados importantes, então toda ação é de responsabilidade do usuário, sendo importante ter atenção redobrada ao manuseá-los.

Mensagens do sistema

Erros podem ocorrer por diferentes motivos, como preenchimento incorreto, falta de dados, problemas no servidor ou desatenção às instruções. Em caso de erro, analisamos a mensagem de erro exibida para identificar a causa e corrigir o problema.

Para garantir a segurança e o funcionamento contínuo do sistema, mensagens de sucesso também são exibidas para confirmar que as ações foram realizadas com sucesso.

Consulta de clientes

Além das ações que alteram os dados do sistema, é igualmente importante contar com uma funcionalidade de consulta de informações dos clientes e seus respectivos equipamentos.

Ao selecionar um cliente na lista gerada e clicar no botão "Consultar Cliente", o sistema exibirá o nome do cliente junto com os detalhes de seus equipamentos cadastrados. Isso permite uma visualização rápida e organizada das informações associadas a cada cliente.
