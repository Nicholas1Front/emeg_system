Ajuda com o controle de estoque

Bem-vindo à página de ajuda!

Aqui você encontrará um guia completo para utilizar o sistema de controle de estoque de forma correta e eficiente.

O que é ? Como funciona ?

O sistema de controle de estoque EMEG é a ferramenta que pode ser utilizada na consulta de todos os itens ou de um item especifíco conforme os mesmos estão listados no banco de dados, este sistema também conta com a adição, exclusão , edição e envio de dados sobre os itens para o servidor .Ele funciona de forma local em seu computador, mas requer conexão com a internet para acessar e sincronizar as informações com a nuvem. Este sistema foi projetado visando suprir a necessidade constante de verificação de peças , ferramentas e materiais gerais que são usados para a manutenção de serviços e do proprio espaço fisíco da empresa.

Opções de ajuda

// Ajuda : Pesquisa de item especifico

Pesquisa de item especifico

Quando este botão é clicado o usuário é levado a uma área de pesquisa e exibição de resultados, onde de acordo com os parametros de pesquisa passado o sistema se encarregará de exibir os itens correspondentes como resultado em uma tabela e se o mesmo não encontrar nenhum resultado irá informar

É recomendado o uso dessa funcionalidade para a execução de uma pesquisa mais refinida buscando uma peça ou ferramenta especifíca que pode ser feita pela própria equipe de manutenção

Preenchimento e pesquisa correta

Como primeiro parâmetro de pesquisa é necessário digitar ou escolher o tipo do item desejado, importante destacar que este sistema possui uma ferramenta de sugestões que conforme esse item ou grupo de itens existir no banco de dados do sistema a mesma irá oferecer opções para autocompletar o campo do tipo de serviço, como mostra a imagem a seguir. Entretanto caso se queira fazer uma pesquisa sem passar o tipo de item é possível usando somente o nome/descrição do item entretanto recomenda-se para sempre especificar o máximo possível a pesquisa

Para o segundo parâmetro de pesquisa temos o nome/descrição do item que conta com o mesmo sistema de sugestão de nome que funciona da seguinte maneira se campo de tipo de item está vazio o programa irá sugerir sugestões através dos nome digitado procurando em todos os itens, entretanto se o campo do tipo estiver preenchido com um tipo como : PEÇA ou FERRAMENTA , as sugestões no campo de nome/descrição aparecerão somente derividas daquele tipo de item.

Passado os parâmetros e clicado no botão de pesquisa o sistema irá fazer a pesquisa de acordo com os parâmtros passados (tipo de item e nome/descrição do item). Se o sistema for bem sucedido na procura pelos itens irá exibir o item ou os itens numa tabela como mostra a imagem a seguir, mas caso não consiga achar nenhum item (seja por digitação erada ou pela não existência deste item ) o programa irá exibir que o item não existe no banco de dados. Importante apontar que a funcionalidade de pesquisa não funciona caso o nome/descrição de serviço esteja vazio

Mensagens do sistema

Em caso de falta de informações ou dados inseridos de maneira incorreta este sistema avisará o usuário por meio de avisos popup que aparecerão na tela. Nestes popups estará contida a mensagem do que esta motivando o erro.

// Ajuda : Mostrar todo estoque

Mostrar todo estoque

Quando este botão é clicado o usuário é levado a uma página de exibição da informação dos itens onde essa página possui dois botões sendo estes : Consultar itens em falta e Consultar itens existente do estoque , onde os mesmos respectivamente mostram as informações dos itens em falta e depois dos itens que o estoque possuí como mostra as imagens abaixo.

O uso dessa funcionalidade é indicado para analisar todos os itens de uma maneira geral para que assim se possa fazer um planejamento mais eficaz na compra de materiais e ferramentas , outra indicação de uso é que essa funcionalidade pode ser util para verificar as alterações que são feitas no sistema como adicionar , exlcuir ou editar arquivos , usar essa funcionalidade pode ser de grande avalia para verificar as informações gerais dos itens e sua conformidade.

// Ajuda : Pesquisa de item especifico

Editar tabela

Quando este botão é clicado o usuário é levado a uma área de adição, exclusão, edição e envio de dados no sistema. Como o proprio nome diz essa sessão do sistema é responsável pela edição e manipulação da tabela por isso deve ser tratada com cautela.

Essa parte do sistema possui quatro botões : Adicionar item ao inventário, Excluir item do inventário, Editar item do inventário e Enviar dados para o servidor

Confirmação por senha

Ao realizar ações como "Adicionar", "Editar", "Excluir" ou "Enviar ao servidor", o sistema solicitará uma confirmação de senha para garantir a segurança e evitar alterações acidentais.

Lembre-se: este sistema serve para gerenciar dados importantes, então toda ação é de responsabilidade do usuário, sendo importante ter atenção redobrada ao manuseá-los.

Adicionar item ao inventário

Quando o botão Adicionar item ao inventário é clicado o usuário é apresentado a uma área para inserir as informações de um novo item onde a mesma possui 4 campos para inserção de dados : tipo do item, nome/descrição do item, estado do item e quantidade

Explicando respectivamente cada campo : o campo do tipo de item diz respeito ao tipo de cada item (peça, ferramenta, material e etc...) e conta com a ferramenta de sugestão de tipos conforme o usuário digita. Em seguida o campo de nome/descrição necessita uma descrição precisa e detalhada do item que será adicionado para evitar divergências entre nomes no sistema do inventário

O campos de estado de item e quantidade diz respeito primeiramente ao estado do item no inventário, se o mesmo existe ou está em falta e caso o item exista no inventário é necessário inserir a quantidade de itens. As imagens abaixo mostram como devem ficar os campos após o preenchimento.

Importante destacar que quanto a estrutura de preenchimento é necessário que se siga como mostrado na imagem para que haja padronização nas nomenclaturas. Outro ponto importante é que nenhum dos campos podem estar vazios pois se o sistema assim detectar irá mostrar uma mensagem de erro . O mesmo vale caso o item seja igual a outro já existe , o sistema retornará uma mensagem informando que o novo item não pode ser igual a um item existente.

Excluir item do inventário

Nesta área do sistema o usuário primeiro é apresentado a dois campos onde serão inseridos o tipo do item e nome/descrição do mesmo, pois como mostra a imagem ao lado o sistema irá buscar a existência do item. Caso as informações inseridas estejam incorretas ou o item não exista no banco de dados o sistema irá retornar uma mensagem de erro informando que o item desejado não foi encontrado ou não existe.

Caso o item inserido exista o sistema irá exibir o mesmo dessa forma que aparece ao lado esquerdo, com todas as informações e dados que o item possui naquele momento e logo em seguida o botão Excluir item estará disponível para confirmar se o usuário realmente deseja excluir aquele item especifíco.

Importante destacar que esta ação irá excluir o item do banco de dados do sistema e deletar todas as suas informações caso o usuário deseje indicar que um item está EM FALTA o mesmo deve se dirigir ao botão Editar item do inventário
