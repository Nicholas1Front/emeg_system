Ajuda com o controle de estoque

Bem-vindo à página de ajuda!

Aqui você encontrará um guia completo para utilizar o sistema de controle de estoque de forma correta e eficiente.

O que é ? Como funciona ?

O sistema de controle de estoque EMEG é a ferramenta que pode ser utilizada na consulta de todos os itens ou de um item especifíco conforme os mesmos estão listados no banco de dados, este sistema também conta com a adição, exclusão , edição e envio de dados sobre os itens para o servidor .Ele funciona de forma local em seu computador, mas requer conexão com a internet para acessar e sincronizar as informações com a nuvem. Este sistema foi projetado visando suprir a necessidade constante de verificação de peças , ferramentas e materiais gerais que são usados para a manutenção de serviços e do proprio espaço fisíco da empresa.

Opções de ajuda

Abaixo selecione a opção na qual você deseja ajuda

// Ajuda : Pesquisa de item especifico

Pesquisa de Item Específico

Quando este botão é clicado, o usuário é levado a uma área de pesquisa e exibição de resultados, onde, de acordo com os parâmetros de pesquisa fornecidos, o sistema exibirá os itens correspondentes como resultado em uma tabela. Caso nenhum item seja encontrado, o sistema informará ao usuário.

É recomendado utilizar essa funcionalidade para realizar uma pesquisa mais refinada, buscando uma peça ou ferramenta específica. Essa pesquisa pode ser realizada pela própria equipe de manutenção.

Preenchimento e Pesquisa Correta

Como primeiro parâmetro de pesquisa, é necessário digitar ou escolher o tipo do item desejado. É importante destacar que este sistema possui uma ferramenta de sugestões que, conforme o item ou grupo de itens existir no banco de dados, oferecerá opções para autocompletar o campo do tipo de serviço. Veja o exemplo na imagem abaixo. Entretanto, caso deseje realizar uma pesquisa sem preencher o tipo de item, é possível usar apenas o nome/descrição do item. Contudo, recomenda-se especificar o máximo possível para obter resultados mais precisos.

Para o segundo parâmetro de pesquisa, temos o nome/descrição do item, que também conta com o mesmo sistema de sugestões. Este sistema funciona da seguinte forma: - Se o campo "tipo de item" estiver vazio, o programa sugerirá itens com base no nome digitado, pesquisando em todos os itens do banco de dados. - Se o campo "tipo de item" estiver preenchido com categorias como PEÇA ou FERRAMENTA, as sugestões no campo de nome/descrição serão limitadas aos itens relacionados a esse tipo específico.

Após definir os parâmetros e clicar no botão de pesquisa, o sistema realizará a busca de acordo com os critérios fornecidos (tipo de item e nome/descrição). Se a pesquisa for bem-sucedida, os resultados serão exibidos em uma tabela, como mostrado na imagem a seguir. Caso contrário, o sistema exibirá uma mensagem informando que o item não existe no banco de dados. Isso pode ocorrer devido a erros de digitação ou à inexistência do item pesquisado. É importante ressaltar que a funcionalidade de pesquisa não funcionará caso o campo "nome/descrição do item" esteja vazio.

Mensagens do sistema

Em caso de falta de informações ou dados inseridos de maneira incorreta este sistema avisará o usuário por meio de avisos popup que aparecerão na tela. Nestes popups estará contida a mensagem do que esta motivando o erro.

// Ajuda : Mostrar todo estoque

Mostrar todo estoque

Quando este botão é clicado, o usuário é levado a uma página de exibição das informações dos itens. Nesta página, há dois botões principais: Consultar itens em falta e Consultar itens existentes no estoque. Esses botões, respectivamente, mostram as informações dos itens que estão em falta e dos itens disponíveis no estoque, como ilustrado nas imagens abaixo.

O uso dessa funcionalidade é indicado para analisar todos os itens de forma geral, permitindo um planejamento mais eficaz na compra de materiais e ferramentas. Outra recomendação de uso é para verificar alterações realizadas no sistema, como adicionar, excluir ou editar itens. Utilizar essa funcionalidade pode ser de grande valor para validar as informações gerais dos itens e garantir a conformidade dos dados no sistema.

// Ajuda : Pesquisa de item especifico

Editar tabela

Quando este botão é clicado o usuário é levado a uma área de adição, exclusão, edição e envio de dados no sistema. Como o proprio nome diz essa sessão do sistema é responsável pela edição e manipulação da tabela por isso deve ser tratada com cautela.

Essa parte do sistema possui quatro botões : Adicionar item ao inventário, Excluir item do inventário, Editar item do inventário e Enviar dados para o servidor

Confirmação por senha

Ao realizar ações como "Adicionar", "Editar", "Excluir" ou "Enviar ao servidor", o sistema solicitará uma confirmação de senha para garantir a segurança e evitar alterações acidentais.

Lembre-se: este sistema serve para gerenciar dados importantes, então toda ação é de responsabilidade do usuário, sendo importante ter atenção redobrada ao manuseá-los.

Adicionar item ao inventário

Quando o botão Adicionar item ao inventário é clicado, o usuário é direcionado a uma área para inserir as informações de um novo item. Esta área possui 4 campos para inserção de dados: tipo do item, nome/descrição do item, estado do item e quantidade.

Explicando cada campo, respectivamente:

- O campo tipo de item refere-se ao tipo do item (por exemplo: peça, ferramenta, material, etc.) e conta com uma ferramenta de sugestões que se ativa conforme o usuário digita.

- O campo nome/descrição do item exige uma descrição precisa e detalhada do item que será adicionado, para evitar divergências nos nomes dentro do sistema de inventário.

O campo estado do item indica se o item existe ou está em falta no inventário. Caso o item exista, o campo quantidade deve ser preenchido com a quantidade disponível. As imagens abaixo ilustram como os campos devem ser preenchidos.

É importante destacar que a estrutura de preenchimento deve seguir o modelo mostrado nas imagens, para garantir a padronização das nomenclaturas. Outro ponto relevante é que nenhum dos campos pode estar vazio. Caso o sistema detecte campos não preenchidos, ele exibirá uma mensagem de erro. Da mesma forma, se o item for igual a um item já existente no inventário, o sistema retornará uma mensagem informando que o novo item não pode ser idêntico ao item já registrado.

Editar item do inventário

A primeira parte da edição de itens começa com a pesquisa da existência do item, para que o usuário possa prosseguir com o processo de edição do item encontrado. Para isso, o usuário é apresentado a dois campos para inserção de dados: tipo do item e nome/descrição do item. Ambos os campos possuem uma ferramenta de sugestões, que exibe tanto os itens existentes quanto os nomes sugeridos, conforme o usuário digita, como mostrado na imagem abaixo.

Após inserir os dados corretamente e clicar no botão "Pesquisar item", o sistema irá realizar a busca com os parâmetros fornecidos. Se o item existir, ele será exibido. No entanto, se o que for inserido for inválido ou se o item não existir, o sistema retornará uma mensagem de erro informando que o item não foi encontrado ou não existe.

Caso o item realmente exista, o sistema exibirá as informações do item e disponibilizará os campos para inserção dos dados, já com as informações existentes. Como mostrado na imagem abaixo, o usuário pode editar o tipo do item, nome/descrição do item, estado do item e quantidade do item. É importante destacar que o item editado não pode ser igual ao anterior. Portanto, é estritamente necessário verificar as novas informações antes de salvar, pois, caso contrário, o sistema retornará uma mensagem de erro informando que as informações estão incorretas.

Excluir item do inventário

Nesta área do sistema, o usuário é inicialmente apresentado a dois campos para inserir o tipo do item e o nome/descrição do item. Como mostrado na imagem ao lado, o sistema buscará a existência do item. Caso as informações inseridas estejam incorretas ou o item não exista no banco de dados, o sistema retornará uma mensagem de erro informando que o item desejado não foi encontrado ou não existe.

Caso o item inserido exista, o sistema exibirá os dados do item, como mostrado na imagem à esquerda, com todas as informações atuais do item. Em seguida, o botão Excluir item estará disponível, permitindo que o usuário confirme a exclusão do item específico.

É importante destacar que esta ação irá excluir permanentemente o item do banco de dados do sistema, apagando todas as informações relacionadas a ele. Caso o usuário deseje apenas indicar que o item está EM FALTA, ele deve utilizar o botão Editar item do inventário.

Enviar alterações para o servidor

Após feita qualquer alteração no sistema, seja Adicionar um item, Excluir ou Editar, é necessário enviar essas alterações para o servidor para que essas mudanças sejam internalizadas no sistema. É importante somente fazer uso deste botão Enviar alterações para o servidor somente quando realmente existerem alterações no sistema, pois se o mesmo detectar que não existem alterações o programa não permitirá prosseguir com a ação de envio retornando uma mensagem de erro.

Realize as alterações necessárias

Com as informações organizadas, faça as alterações no sistema, sempre prestando atenção aos detalhes para evitar a inserção de dados incorretos.

Envie os dados ao servidor

Após finalizar as alterações, clique em "Enviar ao servidor" para sincronizar as informações com a nuvem.

Aguarde o processamento

Depois de confirmar a senha, o sistema precisará de um tempo para processar as informações. Não feche a página até que o processo seja concluído. Uma vez finalizado, você poderá sair com segurança.

Mensagens do sistema

Em caso de falta de informações ou dados inseridos de maneira incorreta este sistema avisará o usuário por meio de avisos popup que aparecerão na tela. Nestes popups estará contida a mensagem do que esta motivando o erro.
