Ajuda com a Criação de Orçamento

Bem-vindo à Página de Ajuda!

Aqui você encontrará um guia completo para utilizar o sistema de criação de orçamento de forma correta e eficiente.

Do que se trata este sistema?

O sistema de criação de orçamento é uma plataforma web focada na criação e administração de orçamentos da EMEG. Este sistema funciona localmente, porém é necessária uma conexão com a internet para acessar as informações e dados dos clientes que estão na nuvem.

Em alguns casos, pode ser necessária a adição de algum cliente ou a edição de um equipamento. Para isso, o sistema de controle de clientes estará disponível e funcional para realizar essas alterações.

Dados para Preenchimento

"Dados para Preenchimento" é o cabeçalho do orçamento, ou seja, as informações exigidas nos campos são necessárias e importantes para identificar o orçamento, tornando-o mais detalhado e preciso tanto para a EMEG quanto para o cliente que solicitou o orçamento.

Abaixo, seguem algumas informações sobre o preenchimento destes campos:

Campos Obrigatórios

Os campos Cliente e Equipamento são obrigatórios. Ou seja, o sistema não aceitará que estes campos estejam vazios; caso contrário, será exibida uma mensagem de erro.

Campos Facultativos

Os campos Data de Entrada, Condição de Pagamento, Garantia e Prazo Estimado não são estritamente obrigatórios. Se deixados em branco, serão exibidos os caracteres "####" no orçamento. No entanto, essas informações também são importantes para que o orçamento seja claro e preciso para o cliente.

Peças e Serviços

Basicamente, o conjunto de itens mais importantes em um orçamento. Neste sistema, está dividido da seguinte forma: Peças que serão ou que foram aplicadas e Serviços que serão ou foram executados.

Ambas, peças e serviços, são adicionados como itens pelo sistema. Esses itens precisam ter Quantidade, Descrição e Valor Unitário.

Se todos os dados estiverem corretos, ao clicar no botão "Adicionar Item", o sistema adicionará as informações, sejam de peças aplicadas ou de serviços executados. Entretanto, se algum dado estiver faltando, o sistema não prosseguirá devido à falta de informações e apresentará uma mensagem de erro.

Quantidade

Para adicionar uma peça ou um serviço, é estritamente necessário descrever a quantidade de peças ou de serviços que estão sendo ou foram aplicados. O formato aceito pelo campo de Quantidade são somente números inteiros.

Descrição

É essencial que a descrição da peça ou do serviço seja coerente e objetiva para evitar qualquer tipo de ambiguidade quanto à natureza da peça ou do serviço. Em especial no caso das peças, em alguns casos é interessante especificar informações como Marca, Código, etc.

Valor Unitário

No campo Valor Unitário, o sistema aceitará dados em formato monetário, ou seja, apenas números e vírgula. O Valor Total é calculado a partir da multiplicação da quantidade pelo valor unitário.

Controle de Itens

Após o botão "Adicionar Item" ser clicado, seja para Peças ou para Serviços, o sistema adicionará essas informações como um item, que ficará visível no orçamento.

Dentro de cada item, existirão dois botões de controle que servem para o manuseio e edição de cada item. A função de cada um desses botões está especificada ao lado.

Ao clicar no botão Editar , os dados do respectivo item serão inseridos de volta nos campos de Quantidade, Descrição e Valor Unitário para edição.

Ao clicar no botão Excluir , as informações do item em questão serão deletadas. Portanto, o uso desta funcionalidade exige atenção redobrada.

Valores Totais do Orçamento

Ao adicionar, remover ou editar um item, o valor total do orçamento é alterado e atualizado em sua exibição. Considerando a estrutura de produção de orçamento, este sistema possui algumas seções que exibem valores totais, sendo elas: Valor Total das Peças Aplicadas, Valor Total dos Serviços Executados e, por fim, Valor Total do Orçamento, que engloba os serviços executados e as peças aplicadas.

Observações Gerais

A seção de Observações Gerais do sistema de criação de orçamentos dedica-se a exibir observações adicionais que podem ser inseridas neste campo. Não é obrigatório seu preenchimento; porém, é sempre importante manter o orçamento o mais preciso e especificado possível.

Gerar Orçamento

Após verificar se todas as informações estão corretas, o botão "Gerar Orçamento" deve ser clicado para que o mesmo seja gerado com êxito. A numeração ou código do orçamento é feita de maneira automática, e as informações anteriormente inseridas nos campos de todo o orçamento são exibidas no documento gerado. É importante pontuar que orçamentos sem nenhum tipo de item (sejam peças ou serviços) podem ser gerados.

Para manter o controle sobre a numeração automática dos orçamentos, foi criado um servidor responsável por receber e processar as numerações de cada novo orçamento. Quando um novo orçamento é gerado, o sistema precisa completar o carregamento dessa numeração antes de exibi-la e enviá-la ao servidor, evitando erros de sincronização. É essencial aguardar a conclusão desse processo para garantir que a numeração correta seja exibida e enviada. Caso ocorra algum problema durante a exibição ou o envio, o sistema exibirá uma mensagem de erro detalhando a causa.

Uma vez gerado o orçamento, são liberadas três opções: Salvar em HTML, Salvar em PDF e Voltar para a edição do orçamento.

Salvar em HTML

Ao clicar neste botão, o orçamento gerado é salvo no formato HTML, que pode ser aberto apenas com um navegador de internet. Recomenda-se verificar todas as informações no documento antes de salvar para evitar alterações desnecessárias.

Salvar em PDF

Como o próprio nome sugere, este botão salva o orçamento no formato PDF, que é mais seguro e confiável para visualização. Recomenda-se primeiro salvar o orçamento em HTML e, somente após isso, salvar em PDF.

Voltar para a edição do orçamento

Caso exista a necessidade de fazer uma alteração em alguma informação no orçamento gerado, este botão servirá justamente para isso. Ao ser clicado, o sistema retorna à página inicial para que o orçamento possa ser editado novamente.

Mensagens do Sistema

Este sistema conta com mensagens de erro ou de atenção que indicam que algo não está correto. No caso deste sistema, as mensagens de erro aparecem quando a lógica do sistema não é seguida corretamente ou quando informações importantes e obrigatórias estão faltando. Além disso o sistema também conta com avisos e mensagens sobre o estado de funcionamento do servidor

Já as mensagens de atenção aparecem pelo mesmo motivo. Apesar de serem informações ou dados importantes, elas podem ficar de fora, pois não irão influenciar negativamente no fluxo do sistema.

De qualquer forma, sempre que o sistema exibir uma mensagem, é necessário que haja atenção e cuidado por parte de quem utiliza o programa para interpretar corretamente qualquer tipo de mensagem que apareça.
