Ajuda com o tabela de preços

Bem-vindo à página de ajuda!

Aqui você encontrará um guia completo para utilizar o sistema de tabela de preços de forma correta e eficiente.

O que é ? Como funciona ?

O sistema de tabela de preço EMEG é a ferramenta que pode ser utilizada na consulta de preços de todos os serviços ou de um serviço especifico e que também pode ser utilizada para adição, exclusão ou edição de serviços na tabela.Ele funciona de forma local em seu computador, mas requer conexão com a internet para acessar e sincronizar as informações com a nuvem. Este sistema foi projetado para ser usado juntamente com o sistema de criação de orçamentos, onde a ideia é que o usuário pesquise o serviço antes de colocar no orçamento.

Opções de ajuda

Abaixo selecione a opção na qual você deseja ajuda

// Ajuda : Pesquisa de item especifico

Pesquisa de item especifico

Quando este botão é clicado o usuário é levado a uma área de pesquisa e exibição de resultados, onde de acordo com os parametros de pesquisa passado o sistema se encarregará de exibir os serviços correspondentes como resultado em uma tabela e se o mesmo não encontrar nenhum resultado irá informar

É recomendado o uso desta funcionalidade no momento de preencher os valores de um serviço em um orçamento, para que haja sempre conformidade entre os orçamentos e os valores descritos na tabela.

Preenchimento e pesquisa correta

Como primeiro parametro de pesquisa, é necessário escolher o tipo de serviço o qual o item inclui, caso esse parametro não seja colocado o programa irá pesquisar entre todos os serviços da tabela. É importante lembrar que esta pesquisa é indicada para achar um item especifico, sendo assim quanto mais especificação melhores serão os resultados como mostrado na imagem abaixo.

Depois de selecionado o tipo de serviço é necessário descrever o serviço desejado como a imagem acima mostra. Este sistema possui uma ferramenta de sugestões onde conforme o usuário digita , e de acordo com o tipo de serviço selecionado, esta ferramenta vai criando sugestões com base nos serviços existentes visando assim facilitar o modo de pesquisa.

Após feita a descrição do serviço que se deseja pesquisar basta clicar no botão de Pesquisa que o sistema irá gerar uma tabela de acordo com os parametros de pesquisa passados : tipo de serviço e descrição do serviço, como mostra a imagem abaixo.Importante apontar que o sistema não fará a pesquisa se a descrição do serviço estiver vazia ! Caso o sistema não encontre o que foi digitado, será informado na tabela que não existem serviços correspondentes.

Mensagens do sistema

Em caso de falta de informações ou dados inseridos de maneira incorreta este sistema avisará o usuário por meio de avisos popup que aparecerão na tela. Nestes popups estará contida a mensagem do que esta motivando o erro.

// Ajuda : Mostrar tabela completa

Mostrar tabela completa

Quando este botão é clicado o usuário é levado a uma área de exibição de resultados mostrando todos os serviços presentes no sistema, e seus respectivos preços e tipos.

É recomendado o uso dessa funcionalidade para analisar todos os serviços de uma maneira geral, outra recomendação para esta funcionalidade é quando os dados do sistema sofrem uma alteração, explicando melhor quando o usuário adiciona um serviço, exclui um serviço ou edita um serviço é recomendado usar esta funcionalidade para verificar tais alterações.

Abaixo um exemplo de como a tabela é disposta ao usuário quando o faz uso do botão de mostrar tabela completa

// Ajuda : Editar tabela

Editar tabela

Quando este botão é clicado o usuário é levado a uma área de adição, exclusão, edição e envio de dados no sistema. Como o proprio nome diz essa sessão do sistema é responsável pela edição e manipulação da tabela por isso deve ser tratada com cautela.

Essa parte do sistema possui quatro botões : Adicionar item a tabela, Excluir item da tabela, Editar item da tabela, Enviar dados para o servidor

Confirmação por senha

Ao realizar ações como "Adicionar", "Editar", "Excluir" ou "Enviar ao servidor", o sistema solicitará uma confirmação de senha para garantir a segurança e evitar alterações acidentais.

Lembre-se: este sistema serve para gerenciar dados importantes, então toda ação é de responsabilidade do usuário, sendo importante ter atenção redobrada ao manuseá-los.

Adicionar item a tabela

Assim que este botão é clicado o usuário será apresentado a 3 campos para inserção de dados relacionados ao item que pretende-se adicionar a tabela

Como a imagem acima mostra, o primeiro campo refere-se ao tipo de serviço que o item terá, o segundo campo refere-se a descrição do serviço que o item terá e por fim o terceiro campo refere-se ao preço/valor que o item terá. Para que o processo de adição do item realmente aconteça nenhum destes campos pode estar vazio

Quanto a forma de preenchimento da descrição do serviço é estritamente necessário que a mesma siga uma estrutura para que as informações sejam padronizadas, abaixo segue a forma de estruturação de preenchimento dos dados e um exemplo do mesmo.

Estrutura de preenchimento

DESCRIÇÃO DO SERVIÇO COMPLETA E RESUMIDA (EQUIPAMENTO REFERENTE)

Exemplo de preenchimento completo

TROCA REPARO CILINDRO DE ELEVAÇÃO (GUINDASTE SANY)

Após todos os campos serem preenchidos corretamente o botão Adicionar item pode ser clicado para dar prosseguimento ao processo de adição. Caso haja algum problema que impeça o sistema de adicionar o item especificado como falta de dados ou item já existente, o mesmo irá retornar uma mensagem de erro apontando onde o erro se encontra.

Excluir item da tabela

Nesta área do sistema existem 2 campos de inserção, com o primeiro campo sendo o tipo de serviço que o item que se deseja excluir possuí, é estritamente necessário selecionar o tipo de serviço pois somente quando um destes é selecionado o campo seguinte é liberado para inserção de dados.

No outro campo temos a descrição do serviço do item que se deseja excluir, este campo conta com um sistema de sugestões da mesma forma que área de pesquisa possuí. Conforme o usuário for digitando as opções com os itens existentes de acordo com a descrição digitada aparecerão para serem clicadas

Por fim ao lado do campo de descrição de serviço existe um visualizador de valor/preço dos serviços que, se o item digitado ou selecionado existir no sistema o visualizador mostrará o valor/preço deste item. Entretanto caso a descrição esteja incompleta ou o item não exista o visualizador não mostrará nenhum valor/preço.

Com todos os campos preenchidos o botão Excluir item pode ser clicado para dar prosseguimento com o processo de exclusão, porém é importante apontar que caso o item informado não exista ou algum dos campos estiver vazio o sistema irá retornar uma mensagem de erro acusando onde o erro ocorrido se encontra.

Editar item da tabela

Assim como na área de exclusão de itens, existem 2 campos para inserção de dados sendo o primeiro campo tipo de serviço que o item que se deseja editar possuí, também é estritamente necessário selecionar o tipo de serviço já que o próximo campo só é liberado quando uma opção é selecionada.

Após o campo de tipo de serviço, temos o campo de descrição do serviço que também conta com um sistema de sugestões que aparecem conforme o que é digitado neste campo. Juntamente com este campo temos o visualizador de preço que foi selecionado ou digitado no campo de descrição corresponder a um serviço existente no sistema, este visualizador mostrará o preço correspondente.

Após a conclusão desses dois campos o botão Próximo passo pode ser clicado e se o que foi inserido nos campos mencionados anteriormente corresponder a um serviço existente o sistema irá liberar a área de edição do item mencionado. Importante citar que se o que estiver nos dois campos não corresponder a nenhum serviço existente o sistema irá retornar uma mensagem de erro informando que nenhum serviço foi encontrado.

Como mostra a imagem acima, após o botão Próximo passo ser clicado e esta outra área ser liberada o usuário é levado a outros três campos de inserção de dados para edição do item mencionado anteriormente. Os campos são semelhantes a os anteriores do sistema : tipo de serviço , descrição do serviço e preço/valor do serviço.

Após serem inseridos os novos dados para edição e atualização do item, o botão Editar item pode ser clicado, entretanto é de suma importância que o novo item não seja igual ao anterior e nem mesmo igual a um item já existente no sistema pois se assim acontecer o sistema irá retornar uma mensagem de erro informando que os dados inseridos para a edição do item estão incorretos e o programa não pode prosseguir com a ação desejada. Se o dados estiverem corretos o sistema irá prosseguir com a edição do item.

Enviar alterações para o servidor

Após feita qualquer alteração no sistema, seja Adicionar um item, Excluir ou Editar, é necessário enviar essas alterações para o servidor para que essas mudanças sejam internalizadas no sistema. É importante somente fazer uso deste botão Enviar alterações para o servidor somente quando realmente existerem alterações no sistema, pois se o mesmo detectar que não existem alterações o programa não permitirá prosseguir com a ação de envio retornando uma mensagem de erro.

Realize as alterações necessárias

Com as informações organizadas, faça as alterações no sistema, sempre prestando atenção aos detalhes para evitar a inserção de dados incorretos.

Envie os dados ao servidor

Após finalizar as alterações, clique em "Enviar ao servidor" para sincronizar as informações com a nuvem.

Aguarde o processamento

Depois de confirmar a senha, o sistema precisará de um tempo para processar as informações. Não feche a página até que o processo seja concluído. Uma vez finalizado, você poderá sair com segurança.

Mensagens do sistema

Este sistema conta com mensagens de erro que indicam que algo não está correto. No caso deste sistema, as mensagens de erro aparecem quando a lógica do sistema não é seguida corretamente ou quando informações importantes e obrigatórias estão faltando. Além disso o sistema também conta com avisos e mensagens sobre o estado de funcionamento do servidor
