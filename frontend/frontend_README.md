# Frontend – Sistema de Gestão da Oficina


## Visão Geral
Interface web para gestão da oficina:
- Clientes
- Equipamentos
- Orçamentos
- Ordens de Serviço
- Anotações diárias
- Assistente de IA (futuro)


## Stack
- React
- Vite
- Axios
- React Query
- React Hook Form
- Zod


## Estrutura
- `pages` → telas
- `components` → componentes reutilizáveis
- `services` → comunicação com API
- `hooks` → lógica reutilizável


## Princípios
- Componentes simples
- Estado previsível
- Formulários validados
- Nenhuma regra de negócio pesada no frontend


## Execução (dev)
```bash
npm install
npm run dev
